Dashboard

     ABMELDEN

    • Angebot erstellen

      Erzeugt ein leeres Angebotsformular.

    • Angebot bearbeiten

      Öffnet ein bereits bestehendes Angebotsformular.

    • Angebot duplizieren

      Erzeugt eine Kopie eines bereits bestehendes Angebotsformulars.

    • Angebot exportieren

      Erzeugt ein druckfertiges Angebot als PDF-Datei aus einem bereits bestehenden Angebotsformular inklusive AGB.

    • Rechnung exportieren

      Erzeugt eine druckfertige Rechnung als PDF-Datei aus einem bereits bestehenden Angebotsformular inklusive AGB und bucht alle Posten.

    • Neues Inventar erstellen

      Erzeugt ein leeres Inventurformular.

    • Inventar bearbeiten

      Öffnet eine Liste aller Vermögensgegenstände und gemieteten Artikel.

    • Dienstleistung erstellen

      Erzeugt ein leeres Dienstleistungsformular.

    • Dienstleistungen bearbeiten

      Öffnet ein bereits bestehendes Dienstleistungsformular.

    • Gesellschafter erstellen

      Erzeugt eine leere Gesellschafterkartei.

    • Firma bearbeiten

      Öffnet die Firmenkartei.

    • Kunde erstellen

      Erzeugt eine neue Kundenkartei.

    • Kunden bearbeiten

      Öffnet eine bereits bestehende Kartei eines Kunden.

    • Jahresabschluss exportieren

      Erzeugt einen druckfertigen Jahresabschluss als PDF aus allen Buchungen.

    • Darlehensrechner

      Erzeugt eine druckfertige Darlehnskalkulation als PDF aus allen Buchungen.

    • Arbeitsstunden hinzufügen

      Erzeugt ein leeres Arbeitsstundenformular.

    • Arbeitsstunden exportieren

      Erzeugt eine druckfertige Übersicht als PDF-Datei aus allen bereits bestehenden Arbeitsstundenformularen.

    • Manager öffnen

      Erzeugt ein Logbuch aller Aktivitäten im DBMS.
